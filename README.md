# moosaabbasi.com — Personal Portfolio

Personal portfolio website built with React, TypeScript, and Three.js. Live at [moosaabbasi.com](https://moosaabbasi.com).

---

## Tech Stack

- **Framework:** React 18 + TypeScript + Vite
- **3D Graphics:** Three.js via `@react-three/fiber` + `@react-three/drei`
- **Animations:** Framer Motion (scroll animations, spring cursor, 3D card tilt)
- **Styling:** Inline styles + CSS custom properties
- **Fonts:** Space Grotesk + JetBrains Mono
- **Deployment:** Vercel (auto-deploy on push)

---

## Sections

- **Hero** — Full-bleed 3D scene with orbiting geometry, particle cloud, and typewriter roles
- **About** — Bio, animated counters, achievement badges
- **Skills** — Tech stack grouped by category
- **Experience** — Vorniqo Solutions · Huawei Technologies · USF Student Government
- **Projects** — 3D tilt cards with GitHub links
- **Contact** — Email + social links

---

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

---

## Deployment

Deployed on Vercel. Any push to `main` triggers an automatic redeploy.

```bash
git add .
git commit -m "your message"
git push
```
